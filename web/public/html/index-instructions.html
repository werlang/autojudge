How to use

First you need to register the problems for me to judge. You need to do this only once for each problem. All data sent is saved on the local storage of your browser.

Click on the Problems button, input a unique name for the problem. Like, for example, crazy-smart-robot-01.

Then you need to upload a zip file containing the input and output files (the test cases) for the problem. Zip all input and output files using the following structure:

    input/
        test00
        test01
        test02
        ...
    output/
        test00
        test01
        test02
        ...

You can name the test cases as you wish, but the names must match between input and output files. The files must be in plain text format. Check here for a sample .zip file that adheres to the structure above.

Next, click on the Choose Files button to select your zip file or drag and drop the file on the designated area. Then click on the Upload button.
