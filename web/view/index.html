Welcome to AutoJudge

AutoJudge is a simple web application that allows you to submit your code and have it automatically judged.

How to use

First you need to submit the input and output files for the problem you want to solve. You need to do this only once for each problem. All data sent is saved on the local storage of your browser.

Zip all input and output files using the following structure:

        input/
            test00
            test01
            test02
            ...
        output/
            test00
            test01
            test02
            ...

You can name the test cases as you wish, but the names must match between input and output files. The files must be in plain text format. Check here for a sample .zip file that adheres to the structure above.
